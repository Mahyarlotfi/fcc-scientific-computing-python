# User Configuration Manager

## Overview

This project was completed as part of the **freeCodeCamp Scientific Computing with Python** certification.

The application provides a simple user configuration manager that allows users to manage application settings.

## Features

- Add a new setting
- Update an existing setting
- Delete a setting
- Display all current settings
- Input validation and user-friendly messages

## Technologies

- Python 3
